# Demonstration DRF within the test task

## Preparing:
`git clone https://github.com/Melhin8/social_demo.git`
`cd social_demo && pipenv install`

## Using:
1. `cd post && ./manage.py migrate && ./manage.py runserver`
2. Run bot `cd ../bot && ./bot.py`.
